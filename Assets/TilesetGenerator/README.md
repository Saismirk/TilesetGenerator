# Tileset Generator

Tileset Generator is a tool for generating tilesets from a set of input sprites or texture.

## Usage

To use Tileset Generator, open the Tileset Generator window from the `Tools/Tileset Generator` menu.

In the Tileset Generator window, you can select the input sprites or texture, and then click the `Generate` button to generate the tileset.

The generated tileset will be saved to the `Assets/TilesetGenerator/Resources/ExportTilemap` folder.
